# nodebotx

TypeScript client for [Express.ms](https://express.ms) [BotX API](https://docs.express.ms) platform.

## Installation

```bash
npm install nodebotx
# or
pnpm add nodebotx
# or
yarn add nodebotx
```

## Quick Start

```typescript
import { BotXClient } from 'nodebotx';

const client = new BotXClient({
  bot_id: 'your-bot-uuid',
  secret_key: 'your-secret-key',
  base_url: 'https://cts.your-company.com',
});

// Send notification to chat
const { sync_id } = await client.notifications.send_direct({
  group_chat_id: 'chat-uuid',
  notification: {
    status: 'ok',
    body: 'Hello from BotX!',
  },
});
```
